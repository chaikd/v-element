# Icon 图标

使用[Fort Awesome](https://fontawesome.com/)作为基础图标。

## 使用图标

  - 需要导入v-icon组件
  - 默认导入所有的图标，搜索图标可从[Fort Awesome](https://fontawesome.com/)查看

## 基本用法
  ``` vue
  <template>
    <v-icon icon="search"></v-icon>
  </template>
  ```
    
  <VIcon icon="search"></VIcon>
